# Micro Apps — Customer Intelligence & CX Tools
A collection of lightweight, AI-assisted tools designed to bridge data, customer experience, and product decision-making.  
These projects focus on turning raw signals and user feedback into clear, actionable insights for Product, Customer Support, and Operations teams.

| Project | Title | Description |
| :-----------: | ----------- | ----------- |
| 1 | <a href='https://github.com/simrandulai/Micro_Apps/tree/main/Customer%20%26%20Account%20Intelligence%20Agent'><u>Customer & Account Intelligence Agent</u></a> | An AI-assisted tool that analyzes customer accounts or target companies to evaluate ICP fit, identify key business signals, assess risks and opportunities, and recommend prioritization and next actions. Designed to support GTM, Customer Success, and Support teams in focusing on high-impact accounts. |
| 2 | <a href='./support-insights-product-cx-engine.md'><u>Product & CX Engine</u></a> | A system that transforms customer support tickets and user feedback into structured insights, including theme clustering, root cause analysis, and actionable product and experience improvements. Built to help teams reduce ticket volume, improve retention, and enhance overall user experience. |
